New & Ambitious Free Contact Form Plugin for Wordpress with File Upload, Captcha, and other useful features. Simple, but flexible!

== Description ==

MagicForm is a Free Wordpress Contact Form Plugin which  supports a number of contact forms simultaneously. Moreover, the form and the contents of the email can be customized with simple markup. It can be integrated with Google Analytics, works with reCAPTCHA, Ajax-powered submitting, offers Drop & Drag Constructor, logs, easy customization, file upload etc.

Features available:
 * Real time drag & drop form builder
 * Required fields and optional fields
 * Field validation and field instructions
 * Form Logs
 * reCAPTCHA
 * Custom autoresponder
 * Custom email notifications
 * Google Analytics tracking
 * UpTime Tracking
 * Sent Mail Auto Checking
 * Magic Contact Forms are compatible with mobile devices

== PRIVACY NOTES ==

In its default configuration, the plugin does not:
 * track users without their knowledge;
 * save users' personal data in a database;
 * send data to external servers (with the exception of UpTime Tracking and Sent Mail functions);
 * use cookies.

== Installation ==

1. Upload the entire `MagicForm` folder to the `/wp-content/plugins/` directory.
1. Activate the plugin through the 'Plugins' menu in WordPress.
You will find MagicForm menu in your WordPress admin panel.
For basic usage, you can also have a look at the [plugin web site](https://magicformplugin.wordpress.com/).
